## Scroll
* Down:                 C-y
* Up:                   C-e
* Half screen up:       C-u
* Half screen down:     C-d

# Delete
* Insert:
** Forward word:  C-d
** Backward word: C-w

* Normal:
** Forward word:        dw
** Backward word:       dvb
** Between bracket:     di**bracket**
** Until next **x**:    dt**x**
** Line:                dd

# Yank
* Current line:     yy
* Current word:     viwy

# Select
* Inside bracket:           vi**bracket**
* Reselect last selection:  gv
* Line:                     V

# Navigation
* Normal:
** Go to char:                           <leader><leader>s
** Move cursor to the middle of screen:  zz
** Go to matching parenthesis:           %

# Swith Case
* Normal:
** Toggle (n) char:      (n)~
** Toggle current word:  g~iw
** Toggle line:          g~~

* Visual:
** Toggle:  U

# Git
* Current file Log: :Glog -- % 
** Commit list:     :copen
